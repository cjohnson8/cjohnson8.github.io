HTML Table Reference

Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML tables
<thead> Table Head The set of rows defining the columb headers in a table
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table row container.
<tfoot> Table Foot The set of rows defining the footer in a table.

Table Attributes

Attribute Name Description
colspan Column Span Defines how many columns a td element should span
rowspan Row Defines how many rows a td element should span

Other HTML Tags

Tag Name Description
<img> Image Inserts an image from source
<!DOCTYPE> Doctype Defines the document type
<a> Anchor Element Defines a hyperlink
<input> Input Defines an input control

CSS Selectors and Specificity

Name Syntax Description
Type Selectors element {style properties} The type selector matches elements by node name. In other words, it selects all elements of the given type within a document
Class Selectors .class_name {style properties} The class selector matches elements based on the contents of their class attribute
ID Selectors #id_value {style properties} The ID selector matches an element based on the value of the elements ID attribute

Common CSS properties

Tag Description
background-color Changes the color of text background
border Creates a border around text
font-family Updates font of text
text-align Aligns text to left, right, or center
